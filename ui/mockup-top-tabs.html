Explain how async/await works in Python with a short example.

async/await is Python's syntax for writing asynchronous code that looks synchronous.

Here's a minimal example:

import asyncio

async def fetch_data():
    await asyncio.sleep(1)
    return {"status": "ok"}

async def main():
    result = await fetch_data()
    print(result)

asyncio.run(main())

The async keyword defines a coroutine, and await pauses execution until the awaited coroutine completes, yielding control back to the event loop.
